# О приложении

Данное приложение предназначено для тестирования личностных типов пользователей. Оно предлагает пользователям пройти тест, состоящий из 25 вопросов с двумя вариантами ответов на каждый вопрос. В результате тестирования пользователь получает информацию о своем типе личности и краткое описание.

Приложение включает в себя следующие функции:
- Регистрация и авторизация пользователей.
- Навигационное меню с тремя страницами: Главная, Тест, Профиль.
- Описание всех типов личностей на главной странице.
- Возможность пройти тест на странице "Тест".
- Отображение результатов тестирования и истории прохождения теста в профиле пользователя.
- Интеграция с внешним API для получения случайной цитаты.


> Это приложение — отличный инструмент для самопознания!
